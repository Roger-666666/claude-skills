### 1️⃣ 深度提示词创作助手 (Prompt Deep Creator)
<details>
<summary><b>点击展开：将模糊想法转化为专业 Prompt 🚀</b></summary>

#### 🔹 功能核心
将零散、口语化、模糊的创作想法，转化为结构化、专业、可直接使用的正式提示词模板。
- **提取要素**：自动识别核心目标、受众、语气风格及约束条件。
- **结构化输出**：生成包含角色设定、任务背景、核心任务及成功标准的标准 Prompt。
- **智能追问**：当输入信息不足时，主动引导补充缺失的关键要素。

#### 🔹 适用场景
- **写作创作**：技术博客、深度长文、科普文章。
- **方案设计**：商业方案、产品需求文档 (PRD)。
- **代码开发**：功能描述、代码审查。

#### 🔹 快速开始
> 文件位置：`prompt-deep-creator.zip`

### 2️⃣ 三省六部 · 多 Agent 思维框架 (Sansheng)
<details>
<summary><b>点击展开：复杂任务的通用行政级协作流 ⚔️</b></summary>

#### 🔹 功能核心
这套框架模拟了古代行政决策体系，将任何复杂任务按照“决策→审议→派发→执行”的严格链条进行闭环处理。
- **太子 · 接旨分拣**：判断任务属性并识别涉及部门。
- **中书省 · 起草方案**：拒绝空话，制定详细且可执行的子任务清单。
- **门下省 · 审议把关**：**核心特色**，不合格的方案会被直接“封驳”打回重做。
- **六部 · 实际执行**：兵部(代码)、礼部(文档)、户部(数据)等部门产出真实成果。

#### 🔹 技术特性
- **双模式运行**：支持通过 Claude CLI 进行真实的 LLM 协作，也支持纯框架模式指引。
- **Web 看板**：支持启动 `serve.py` 开启可视化任务监控界面。

#### 🔹 快速开始
> 文件位置：`The Three Provinces and Six Ministries System AI Agent (win)_skill_v1.0.zip`
</details>
