## 🌐 Language Switching

### Method 1: Using the Switch Script

```bash
# Switch to English
./switch_lang.sh en

# Switch to Chinese
./switch_lang.sh zh
```

### Method 2: Manual File Selection

- **English**: Use `README.md`, `SKILL.md`, `scripts/edict_en.py`
- **Chinese**: Use `README_CN.md`, `SKILL_CN.md`, `scripts/edict.py`

## 📦 Installation

### For Claude CLI

1. Copy the skill folder to your Claude CLI skills directory:
   ```bash
   # Example path (may vary)
   cp -r ./skills/sansheng ~/.gemini/antigravity/scratch/skills/
   ```

2. Or configure in `skills.json`:
   ```json
   {
     "skills": [
       {
         "name": "sansheng",
         "path": "./skills/sansheng/SKILL.md"
       }
     ]
   }
   ```

### For Other Tools

Follow the installation instructions in the respective documentation files.
